# Dice

## Summary

A web client and WebSockets server for playing dice.

## Requirements

python3{,-websockets}

Included in libpython3:
	argparse, asyncio, datetime, http, json, logging, os, random, ssl

## Test

### Client

cd client
make test

### Server

cd server
make test

## Build

### Client

cd client
make clean
make publish
make rsync_upload

### Server

cd server
make clean
make deb
